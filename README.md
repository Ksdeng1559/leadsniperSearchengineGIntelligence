# LeadSniper Search Growth Intelligence

A revenue-focused SEO, local-search, and AI-visibility application for identifying growth opportunities among local service businesses.

## Planned capabilities

- Free domain and local search audit
- DataForSEO-powered keyword and competitor intelligence
- Commercial opportunity scoring
- SEO, AEO, and local visibility analysis
- LeadSniperAI growth-opportunity qualification
- Revenue scenarios and prioritized action plans
- Secure server-side integrations and CRM routing

## Status

Initial application development is in progress.
